# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it by:

1. **Email**: Create an issue with the label `security` (do not include sensitive details publicly)
2. **GitHub Security Advisory**: Use the "Report a vulnerability" button in the Security tab

We will respond to security reports within 48 hours.

## Supported Versions

This is a documentation repository for GitHub Foundations training materials. We maintain:

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Security Best Practices

This repository contains:
- Educational content and study materials
- External links to Microsoft Learn and GitHub documentation
- No executable code or sensitive data

Please report any:
- Broken or malicious links
- Outdated security-related information
- Misleading or incorrect security guidance

## Response Process

1. We will acknowledge your report within 48 hours
2. Investigate and validate the issue
3. Fix the issue and update relevant documentation
4. Credit reporters (unless anonymity is requested)

Thank you for helping keep this educational resource safe and accurate!
